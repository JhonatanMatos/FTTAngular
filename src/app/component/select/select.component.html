Parametros do FTTSelectEnums

OnItemSelected: Aqui podemos adicionar a chamada de um metodo ao trocar o item
BorderRight: Adiciona uma borda a direita
bind-Value: Definimos a variavel que vai receber o dado
T: Definimos o enum que vai montar o DDL
Size: Tamanho do componente na tela
Required: Se é obrigatorio preencher o componente ao dar um submit
Label: Texto do label
IsReadOnly: Desabilita o componente
